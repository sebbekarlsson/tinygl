# Tiny OpenGL Example
> A tiny OpenGL example of a bouncing triangle.

## Build & Run
> First, run:
```bash
make glad
```
> _You only need to run this if this is the first time you are compiling it_.  
> Secondly, to compile, run:
```bash
make
```
> And to run it:
```bash
./a.out
```
> Here is it all together:
```bash
make glad && make && ./a.out
```
